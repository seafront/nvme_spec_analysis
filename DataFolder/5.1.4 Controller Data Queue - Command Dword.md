**Figure 162: Controller Data Queue – Command Dword 10**

|Bits|Description|
|---|---|
|31:16|**Management Operation Specific (MOS):** The definition of this field is specific to a management operation.|
|15:08|Reserved|
|07:00|**Select (SEL):** This field specifies the type of management operation to perform.|

**Select Values**

|Value|Management Operation|
|---|---|
|0h|Create Controller Data Queue|
|1h|Delete Controller Data Queue|
|2h to BFh|Reserved|
|C0h to FFh|Vendor Specific|

|Bits|Description|
|---|---|

|15:01|Reserved|
|00|**Physically Contiguous (PC):** If set to '1', the Controller Data Queue is physically contiguous. If cleared to '0', the Controller Data Queue is not physically contiguous.|
