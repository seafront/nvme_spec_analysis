**Figure 198: Get Log Page – Command Dword 10**

|Bits|Description|
|---|---|
|31:16|**Number of Dwords Lower (NUMDL):** This field specifies the least significant 16 bits of the number of dwords to return.|
|15|**Retain Asynchronous Event (RAE):** This bit specifies whether to retain or clear an Asynchronous Event.|
|14:08|**Log Specific Parameter (LSP):** If not defined for the log specified, this field is reserved.|
|07:00|**Log Page Identifier (LID):** This field specifies the identifier of the log page to retrieve.|
