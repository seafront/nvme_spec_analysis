**Figure 157: Capacity Management – Command Dword 10**

|Bits|Description|
|---|---|
|31:16|**Element Identifier (ELID):** This field contains a value specific to the value of the Operation field.|
|15:04|Reserved|
|03:00|**Operation (OPER):** Specifies the operation to be performed by the controller.|

**Operation Values**

|Value|Definition|Element Identifier Field|
|---|---|---|
|0h|Select Capacity Configuration|Capacity Configuration Identifier|
|1h|Create Endurance Group|Domain Identifier|
|2h|Delete Endurance Group|Endurance Group Identifier|
|3h|Create NVM Set|Endurance Group Identifier|
|4h|Delete NVM Set|NVM Set Identifier|
|5h to Fh|Reserved||
